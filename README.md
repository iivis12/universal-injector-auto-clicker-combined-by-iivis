# Minecraft Auto Clicker & Injector (Single Player)

## ⚠️ Disclaimer
This project is intended for **educational purposes and single-player use only**.  
Do NOT use this in multiplayer environments or to gain unfair advantages.

---

## 📌 Overview
This project provides a simple injector and auto-clicker designed for Minecraft single-player worlds.  
It can be used for testing mechanics, automation, and learning purposes.

---

## 🚀 Features
- 🖱️ Adjustable auto-click speed (CPS)
- ⏱️ Toggle on/off with a hotkey
- 💉 Basic injector functionality
- 🎮 Works in Minecraft single-player mode
- ⚙️ Lightweight and easy to use
